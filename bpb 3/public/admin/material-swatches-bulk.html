Upload an entire product's color swatches at once

Drag a folder of swatch images (downloaded from belgard.com) and this tool auto-matches each image to a catalog variant by color, then uploads all of them in parallel. Faster than one-at-a-time for products with 10+ variants.

How this works:
  1. Download all color swatches for one product from belgard.com into its own folder (e.g. Dimensions™ Pavers _ Belgard/)
  2. Drop the entire folder into the zone below
  3. Tool detects the product family from folder name, parses each image filename to extract color name, and matches to catalog variants
  4. Review the preview grid, then click Upload all
  5. Each image uploads once, but writes to every catalog row that shares that color (e.g. Dimensions 12 Scandina Gray + Dimensions 18 Scandina Gray get the same swatch)
  6. Variants that already have a swatch are skipped automatically
