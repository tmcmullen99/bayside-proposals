Replace material hero images

Update the lifestyle photo shown on each material card across the homeowner catalog, designer editor, and future proposal pages. Drag a new image onto any card to replace it instantly.

How it works: Uploaded images are stored at a deterministic URL — replacing the same material's image overwrites the existing file. The catalog, homeowner accounts, and any newly-built proposals will pick up the new image automatically. Existing published proposal pages keep the image they were built with (this is intentional — they're frozen snapshots).
